## 📖 Sobre el Proyecto

Este proyecto fue desarrollado como prototipo para la Hackathon. Busca resolver el desafío de la retención de usuarios en aplicaciones de telecomunicaciones mediante la integración de mecánicas de gamificación, descubrimiento local y transacciones rápidas. 

En lugar de puntos, los usuarios reciben monedas canjeables al consumir en los comercios aliados de su barrio.

## ✨ Características Principales (Features)

* 📱 **Feed de Descubrimiento:** Una interfaz de *scroll* vertical inmersiva (estilo TikTok) para explorar promociones y restaurantes locales con interacciones sociales (Me gusta, Comentarios).
* 🗺️ **Mapa Interactivo:** Geolocalización de tiendas cercanas donde los usuarios pueden cumplir misiones físicas (Ej. "Compra 10 Bs y gana 50 monedas").
* 💳 **Billetera & Escáner QR:** Módulo transaccional de acceso rápido para simular pagos y reclamar recompensas directamente en el local.
* 🎁 **VIVA Days (Eventos):** Pop-ups y módulos de escasez (ofertas por tiempo limitado) para canjear productos de la Fundación VIVA.

---

## 🛠️ Stack Tecnológico

El prototipo está construido priorizando una interfaz fluida (Mobile-First) y alta disponibilidad para demostraciones:

* **Framework:** Next.js (React)
* **Estilos:** Tailwind CSS
* **Iconografía:** Lucide React
* **Gestión de Datos:** Simulada mediante JSON.
